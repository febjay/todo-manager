# TODO Менеджер

Консольное приложение для управления задачами с сохранением в JSON.

## Возможности
- Добавление задач с автоматической нумерацией
- Отметка задач как выполненных
- Удаление задач
- Сохранение в JSON файл
- Загрузка задач при запуске

## Установка и запуск
```bash
python todo_manager.py
Структура данных
Каждая задача хранится в виде словаря:

json
{
    "id": 1,
    "text": "Купить хлеб",
    "done": false
}
text
